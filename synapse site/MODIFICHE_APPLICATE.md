# Modifiche applicate

- Aggiunte sezioni dinamiche gestibili da admin: Siti web, Altri servizi, Recensioni animate.
- Aggiunto listino completo dei pacchetti sito web e servizi extra.
- Aggiunta chiusura chat da admin con classificazione Risolto / Non risolto.
- Aggiunta presenza live nel pannello admin: utenti online, pagina/sezione vista, ultima attività.
- Aggiunta gestione creazione account admin dal pannello admin.
- Migliorato pannello segnalazioni con nome account e flusso chat più chiaro.
- Aggiunta clausola sconti e rafforzata politica no rimborso / no recesso.
- Aggiunte animazioni leggere e compatibili con prefers-reduced-motion, senza cambiare l'identità visiva del sito.
- Migliorata registrazione con requisito password visibile: almeno 8 caratteri.

## Avvio

Questo pacchetto non include node_modules. Dopo l'estrazione eseguire:

```bash
npm install
npm start
```

Il database nella cartella `data/` è stato preservato nel pacchetto.
